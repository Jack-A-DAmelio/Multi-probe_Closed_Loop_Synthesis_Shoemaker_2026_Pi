# Multi-probe_Closed_Loop_Synthesis_Shoemaker_2026_Pi This code supports the implementation of a mutliple probes to record chemical descriptors of solution growth of bulk crystals. Measurements include temperature, volume, pH, and imaging the solution. This accomplished on a two-machine network between a PC and Raspberry Pi.
